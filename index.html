Execution States

  • WAIT — structure exists, no confirmation. No entry.
  • PENDING — valid setup, waiting trigger. Use pending orders.
  • ACTIVE — confirmation broken, trade live. Manage only.
  • MANAGE — deep in profit. No fresh entry without new structure.

These are analyst-axis labels — never put them in the fresh-entry verdict field.

Confirmation First

If a confirmation level exists on the chart, the entry is a pending order at that level. Never enter early. Near trigger ≠ trigger. "Watch and see" from the analyst = PENDING at best, often WAIT.

Late Entry Filter

If price has moved far from optimal entry → TOO LATE. Confidence downgraded. Chasing not justified. Pair then receives a RE-ARM evaluation — every TOO LATE pair gets either ARMED or NO VALID ZONE. See RE-ARM Protocol card.

Risk Model

Fixed €100 per trade. Lot size auto-derived from SL pip distance (see auto-lot strip). Total portfolio exposure < 4.5%. Below-gate RE-ARM entries use halfRisk (€50). Always confirm lots on MT5 before executing.

Duplicate Prevention

No automatic re-entry on an already-traded pair unless: new structure, new RR, retrace resets entry quality, or trade moved deep in profit. RE-ARM ARMED is the controlled exception — passive limit, full RR gate enforced.

Analyst Intent Override

"Stay out", "wait", "corrective", "risky", or any reset with no trigger → verdict downgraded to NO TRADE regardless of chart attractiveness. Latest PDF always overrides older PDFs.

Confidence Bands

  • 90%+ — elite conviction, full size
  • 80–89% — high confidence, full size
  • 70–79% — decent setup, standard size
  • 60–69% — caution, reduce size
  • <60% — skip or paper-trade only

Invalidation Is Hard

Price crossing the invalidation level closes the trade rationale entirely. No exceptions, no re-counting hope. DATA HOLD: pair is NO TRADE until levels verified from source PDF.

Reading the Arrow

Use the arrow's tail (origin), not its shape. Tail near current price = move launching now. Price far along the arrow = TOO LATE.

Straight vs bent = drawing artifact, NOT a signal. Entry decided by labeled confirmation/invalidation levels only.

RE-ARM Protocol

Every TOO LATE pair gets RE-ARM evaluation: ARMED = analyst-drawn level with RR ≥ 2:1 — place passive limit only. BELOW GATE = zone exists but RR < 2:1 — skip or half-risk override. NO VALID ZONE = no analyst level — skip.

Passive limits only. Missing the move when price doesn't pull back = the system working correctly.

Schema Guard

Runs on every page load. Checks: (1) fresh verdict in enum; (2) exactly 4 level tuples per pair; (3) SELL: SL > entry; BUY: SL < entry; (4) ARMED RE-ARM clears 2:1 RR gate.

Red banner at top lists all violations. Do not trade a flagged pair until verified against source PDF.
